The script parse_iDigBio_Occurrences.pl parses an occurrence.csv file provided by iDigBio by extracting the following fields: ID (iDigBio core id), latitude, longitude, date of collection (as a decimal), and day of year of collection. It also removes specimens with georeference coordinates outside of a provided bounding box. This information is provided in a file; PodophyllumPeltatum.info.txt provides a template.
